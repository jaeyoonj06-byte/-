# 논문 도우미 (Thesis Helper)

> 대학생이 논문을 더 쉽고 정확하게 작성할 수 있도록 **템플릿, 자동 인용, 형식 검사**를 제공하는 웹 기반 논문 작성 지원 플랫폼

![Thesis Helper](https://img.shields.io/badge/version-1.0.0-blue) ![License](https://img.shields.io/badge/license-MIT-green) ![Status](https://img.shields.io/badge/status-active-brightgreen)

---

## 🎯 개요

논문 작성 과정에서 겪는 **형식 작성의 어려움**을 해결하기 위해 만들어진 서비스입니다. 대학생들이 논문의 인용 형식, 참고문헌 작성, 템플릿 적용에 더 이상 헷갈리지 않도록 돕습니다.

### 핵심 가치
- ✅ **시간 절약**: 자동 각주·참고문헌 생성으로 수작업 최소화
- ✅ **형식 정확성**: 한국연구재단, APA, MLA, Chicago 4가지 표준 지원
- ✅ **학습 지원**: 단계별 논문 작성 가이드 제공
- ✅ **즉시 사용**: 다운로드 가능한 .docx 템플릿 3종 제공

---

## 🚀 주요 기능

### 1️⃣ 작성 가이드 (`/guide`)
논문 작성의 모든 단계를 체계적으로 안내합니다.

**포함 내용:**
- 📋 논문 구성 (표지, 목차, 초록, 서론, 본론, 결론, 참고문헌)
- 📝 초록(Abstract) 작성법 (200~300자 요약 기법)
- 🔍 서론(Introduction) 작성법 (연구 배경, 문제 제시)
- 🔬 연구방법(Methodology) 작성법 (연구 설계, 데이터 수집)
- 📚 참고문헌 작성법 (인용 형식별 작성 규칙)

**특징:**
- 사이드바 목차로 빠른 탐색
- 모바일 탭 인터페이스
- 섹션 간 네비게이션

---

### 2️⃣ 각주·참고문헌 생성기 (`/citation`)
자료 정보를 입력하면 **자동으로 각주와 참고문헌**을 생성합니다.

#### 📥 입력 형식

**단행본(Book):**
```
저자: 김철수
제목: 한국 현대사 연구
출판사: 서울 출판사
발행연도: 2024
```

**학술논문(Journal Article):**
```
저자: 이영희
제목: 기후변화와 농업 생산성의 관계
학술지: 농업경제학회지
발행연도: 2023
```

**인터넷 기사(Web Article):**
```
저자: 박민준
제목: 대학 등록금 인상 논란
출처: 뉴스 매체명
발행연도: 2024
```

#### 📤 출력 예시

**한국연구재단 형식:**
```
1) 김철수, 『한국 현대사 연구』, 서울 출판사, 2024.
2) 이영희, "기후변화와 농업 생산성의 관계", 『농업경제학회지』, 2023.
3) 박민준, "대학 등록금 인상 논란", 뉴스 매체명, 2024.
```

**APA 7판 형식:**
```
Kim, C. S. (2024). 한국 현대사 연구. Seoul Publishing House.
Lee, Y. H. (2023). 기후변화와 농업 생산성의 관계. 농업경제학회지.
Park, M. J. (2024). 대학 등록금 인상 논란. News Media.
```

**MLA 9판 형식:**
```
Kim, Chul-soo. 한국 현대사 연구. Seoul Publishing House, 2024.
Lee, Young-hee. "기후변화와 농업 생산성의 관계." 농업경제학회지, 2023.
Park, Min-jun. "대학 등록금 인상 논란." News Media, 2024.
```

**Chicago 17판 형식:**
```
Kim, Chul-soo. 한국 현대사 연구. Seoul Publishing House, 2024.
Lee, Young-hee. "기후변화와 농업 생산성의 관계." 농업경제학회지 (2023).
Park, Min-jun. "대학 등록금 인상 논란." News Media, 2024.
```

#### 🎯 기능
- 📌 **자동 정렬**: 저자명 기준 가나다순(한글) / 알파벳순(영문)
- 📋 **형식 선택**: 4가지 인용 형식 즉시 변환
- 📋 **복사**: 전체 참고문헌 목록 한 번에 복사
- 💾 **다운로드**: .txt 또는 .docx 형식으로 저장

---

### 3️⃣ 논문 템플릿 (`/templates`)
학술 기준에 맞춘 **3가지 논문 템플릿**을 즉시 다운로드합니다.

| 템플릿 | 글꼴 | 크기 | 줄간격 | 여백 | 용도 |
|--------|------|------|--------|------|------|
| **학술 논문** | Batang | 12pt | 160% | 25mm | 학술발표, 학회 제출 |
| **졸업 논문** | Shin Myeongjo | 11pt | 180% | 30mm | 학위 논문, 대학원 제출 |
| **학술 보고서** | Malgun Gothic | 11pt | 160% | 25mm/30mm | 과제 보고서, 연구 보고서 |

**다운로드 후:**
1. Microsoft Word에서 .docx 파일 열기
2. 제목, 저자명, 내용 입력
3. 참고문헌 생성기에서 만든 목록 붙여넣기
4. 완성된 논문 저장

## 📱 사용 예시

### 시나리오: 졸업논문 작성

#### 1단계: 가이드 학습
```
1. "작성 가이드" 클릭
2. 사이드바에서 "논문 구성" 선택
3. 표지, 목차, 초록 작성법 학습
4. "다음" 버튼으로 "초록 작성법" 이동
```

#### 2단계: 템플릿 다운로드
```
1. "템플릿" 페이지 이동
2. "졸업 논문" 템플릿 카드 선택
3. ".docx 다운로드" 버튼 클릭
4. 다운로드된 파일을 Word에서 열기
```

#### 3단계: 자료 입력 및 참고문헌 생성
```
1. "인용 생성기" 페이지 이동
2. 자료 유형 선택 (단행본/학술논문/인터넷 기사)
3. 저자명, 제목, 출판사 등 입력
4. "목록에 추가" 클릭
5. 형식 선택 (한국연구재단 / APA / MLA / Chicago)
6. 생성된 참고문헌 복사
7. 템플릿 파일에 붙여넣기
```

## 🔧 로컬 개발 설정

### 필수 요구사항
- Node.js 22.13.0+
- pnpm 10.4.1+

### 설치 및 실행

```bash
# 1. 프로젝트 클론
git clone <repository-url>
cd thesis-helper

# 2. 의존성 설치
pnpm install

# 3. 개발 서버 시작
pnpm dev

# 4. 브라우저에서 열기
# http://localhost:3000
```

### 빌드 및 배포

```bash
# 프로덕션 빌드
pnpm build

# 빌드 결과 확인
pnpm preview
```

## 🎓 사용 팁

### 각주 형식 선택 가이드

| 형식 | 사용 분야 | 특징 |
|------|---------|------|
| **한국연구재단** | 국내 학술지, 학위논문 | 한글 친화적, 가장 일반적 |
| **APA 7판** | 심리학, 교육학, 사회과학 | 저자-연도 방식, 국제 표준 |
| **MLA 9판** | 인문학, 문학, 언어학 | 저자-쪽수 방식, 영문 중심 |
| **Chicago 17판** | 역사학, 철학 | 각주/미주 방식, 상세 인용 |

### 자료 입력 시 주의사항

✅ **올바른 예시:**
- 저자: 김철수 (또는 Kim, Chul-soo)
- 제목: 정확한 원문 그대로
- 출판사: 출판 도시 포함 (예: 서울, New York)
- 연도: 4자리 숫자 (예: 2024)

❌ **피해야 할 입력:**
- 저자: "김철수 외" (대표 저자만 입력)
- 제목: 약자나 오타 포함
- 출판사: 불완전한 정보
- 연도: "최근" 또는 "미정" 같은 모호한 표현
